# Análise de Rentabilidade Operacional
## MEI — Logística de ultima milha - Shopee

### Sobre o projeto
Análise financeira completa de operação de entregas 
para a Shopee, cobrindo o período de Dez/2025 a Jan/2026.

## Ferramenta utilizada
- Microsoft Excel (fórmulas avançadas, tabelas dinâmicas)
- Power BI (em desenvolvimento)

### Principais indicadores

| INDICADOR | | VALOR |
|---|---|
| Receita bruta total | R$ 5.080,90 |
| Margem operacional | 86,5% |
| Margem Liquída | 71,1% |
| Total de entregas | 1.010 |
| Índice de eficiência por km | 86,5% |

### Estrutura da análise
- Base de dados diária com 19 dias de operação
- DRE simplificada com deduções tributárias e financeiras
- Composição de receita por fonte
- Composição de despesas operacionais
- Análise de eficiência por km rodado

### Principais insights
1. Combustível representa 71,4% do custo operacional
2. Margem líquida de 71,1% após DAS e financiamento
3. Melhor dia: 23/01 com R$78,00 de receita por entrega
4. Índice de eficiência de 86,5% por km rodado
